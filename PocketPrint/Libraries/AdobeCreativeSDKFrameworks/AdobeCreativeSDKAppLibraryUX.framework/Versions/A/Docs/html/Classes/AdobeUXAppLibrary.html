Inherits from NSObject

Overview

Core class for adding the Adobe Creative Cloud App Library to your application. This creates a way for folks to browse, and install, other applications that make use of the Creative SDK.

The following AdobeAppLibraryCategories constants can be used to setup the inclusionList and filter arrays:

  • kALCategoryAudio
  • kALCategoryCapture
  • kALCategoryCommunity
  • kALCategoryDesign
  • kALCategoryIllustration
  • kALCategoryPhotos
  • kALCategoryPresentation
  • kALCategoryProductivity
  • kALCategorySocial
  • kALCategoryStorage
  • kALCategoryVideo
