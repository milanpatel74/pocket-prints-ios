AdobePublishCommentPanelButton Class Reference

Inherits from UIButton
Declared in AdobePublishCommentPanelButton.h

Overview

AdobePublishCommentPanelButton can be used to present an AdobePublishCommentPanel. It reacts to unread comment notifications, and displays the unread comment count.
