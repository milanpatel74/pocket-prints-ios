showIn:fromView:fromRect:permittedArrowDirections:

Invoking this method displays the Share Menu in a pop-up in the provided view controller’s view using the rectangle region as location of the button that the pop-up should point to, along with the permitted arrow directions for the popover.

- (void)showIn:(UIViewController *)viewController fromView:(UIView *)view fromRect:(CGRect)rect permittedArrowDirections:(UIPopoverArrowDirection)arrowDirections

Parameters

The view controller for the share menu.

arrowDirections

Specifies in what direction the share menu can be displayed.
