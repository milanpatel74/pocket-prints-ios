AdobeUXAssetBrowserOption

Options that control the various features and functionalities of the Asset Browser.

Definition

typedef NS_OPTIONS(NSUInteger, AdobeUXAssetBrowserOption ) {
   EnableMultiSelect = 1,
   ShowMultiSelectOnPopup = 1 < < 1,
   SortAlphabeticallyOnPopup = 1 < < 3,
   EnablePSDLayerExtraction = 1 < < 4,
   EnableMultiplePSDLayerSelection = 1 < < 5,
};

Constants

EnableMultiSelect

Enable multiple assets to be selected in the Asset Browser. You’ll need to also specify the ShowMultipleSelectOnPopup for this option to take effect.

Declared In AdobeUXAssetBrowserConfiguration.h.

ShowMultiSelectOnPopup

Whether to enable the multiple asset selection mode in the Asset Browser. This option only works in conjunction with the EnableMultiSelect option. Specifying this option alone has no effect.

Declared In AdobeUXAssetBrowserConfiguration.h.

SortAlphabeticallyOnPopup

By default the assets are sorted by recently updated date. Specify this option so the assets are sorted alphabetically. NOTE: This option only applies to the Files (AdobeAssetDataSourceFiles) datasource.

Declared In AdobeUXAssetBrowserConfiguration.h.

EnablePSDLayerExtraction

Enable layer extraction for PSD files. To allow multiple layers and/or layer groups to be selected, specify the EnableMultiplePSDLayerSelection option as well.

Declared In AdobeUXAssetBrowserConfiguration.h.

EnableMultiplePSDLayerSelection

Whether to allow multiple PSD layer selection when exploring a PSD file’s layers. If EnablePSDLayerExtraction isn’t specified, this option does nothing.

Declared In AdobeUXAssetBrowserConfiguration.h.
