Overview

AdobeCommunityCategory represents a category in the Adobe Community. It provides methods that allow the client to obtain information about the category and to enumerate the categories available in the Creative Cloud Community.

categoriesForCommunityID:priority:successBlock:errorBlock:

Fetch all the available categories from the server asynchronously.

+ (void)categoriesForCommunityID:(NSString *)communityID priority:(NSOperationQueuePriority)priority successBlock:(void ( ^ ) ( NSArray *categories ))successBlock errorBlock:(void ( ^ ) ( NSError *error ))errorBlock

Parameters

communityID

The ID of the community.

priority

The priority of the network request.

successBlock

Block to execute when the categories have been retrieved. An NSArray of AdobeCommunityCategory objects will be passed to that block.

errorBlock

Block to execute if an error occurs. Specify @c NULL to omit.

Discussion

Fetch all the available categories from the server asynchronously.
