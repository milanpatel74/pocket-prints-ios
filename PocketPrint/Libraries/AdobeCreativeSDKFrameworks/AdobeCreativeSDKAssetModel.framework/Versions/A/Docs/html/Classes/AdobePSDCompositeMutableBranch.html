removeMaskOf:

Removes the layer mask of the given layer node. Is a no-op if the layer node doesn’t have a mask.

- (AdobePSDLayerNode *)removeMaskOf:(AdobePSDLayerNode *)layer

Parameters

layer

The layer node to remove the mask from.

Return Value

The updated layer node or nil if something went wrong.

Discussion

Removes the layer mask of the given layer node. Is a no-op if the layer node doesn’t have a mask.

Note: This method returns the modified layer node but doesn’t modify the layer node that was passed in. If you want to make further changes to the layer node you will need to use the returned layer node to do so.
