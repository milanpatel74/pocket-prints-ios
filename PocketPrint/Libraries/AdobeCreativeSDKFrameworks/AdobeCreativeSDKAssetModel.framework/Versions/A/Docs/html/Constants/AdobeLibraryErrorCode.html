AdobeLibraryErrorSyncNotAvailableDueToUserSession

AdobeLibraryManager cannot sync because the manager is suspended due to an authentication failure. This is because either no valid user session has been passed during startup or the user session has expired after the manager has been started. Clients should call AdobeLibraryManager#unsuspendSyncDueToNewAuthenticationStatus to unsuspend the manager after they are sure the storageSession passed to the manager has now a valid authentication token.
