initWithDCXSwatches:

Initializes with an array of DCX swatches. Each swatch is expected to be an NSDictionary.

- (instancetype)initWithDCXSwatches:(NSArray *)dict

Parameters

dict

Dictionary of DCX Swatches used to initialize the ColorTheme.

Discussion

Initializes with an array of DCX swatches. Each swatch is expected to be an NSDictionary.

For example, the following two lines are equivalent: @code

[[AdobeColorTheme alloc] initWithDCXSwatches@[@{@“value”: @{@“r”: @“0.5”, @“g”: @“1.0”, @“b”: @“0.25”}}]]; [[AdobeColorTheme alloc] initWithUIColors:@[[UIColor colorWithRed:0.5 green:1.0 blue:0.25 alpha:1.0]]]; @endcode

We currently support only the RGB color model.
