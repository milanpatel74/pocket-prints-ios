Inherits from AdobeAssetFile : AdobeAsset : NSObject
Declared in AdobeAssetPSDFile.h

Overview

AdobeAssetPSDFile represents a Photoshop Document (PSD file). It provides methods for reading PSD previews and composites from PSD files, creating PSD files from composites, and getting layer renditions from PSD files.

An array of AdobeAGCImageComponent objects. Only image/jpg and image/png are supported content types.
