Overview

AdobePhotoCollection is a container of collections in a user’s catalog. Each collection contains zero or more AdobePhotoAssets.

- (void)listAssetsOnPage:(AdobePhotoPage *)page withSortType:(AdobePhotoCollectionSort)sortType withLimit:(NSUInteger)limit withFlag:(AdobePhotoCollectionFlag)flag onCompletion:(void ( ^ ) ( AdobePhotoAssets *assets , AdobePhotoPage *previousPage , AdobePhotoPage *nextPage ))successBlock onError:(void ( ^ ) ( NSError *error ))errorBlock

Parameters

page

The page to pass in, use nil to start from the beginning.

sortType

The sorting option.

limit

The number of assets to return. Default value is 100, max is 500. Please note that the response may contain more than ‘limit’ number of assets returned if the assets at the ‘limit’ boundary has the same capture_date.

flag

The result to return the assets using.

successBlock

Get a list of all AdobePhotoAssets for the logged in user if successful.

errorBlock

Optionally be notified of an error.

Discussion

List the assets for this collection in the Adobe Photo service asynchronously.
