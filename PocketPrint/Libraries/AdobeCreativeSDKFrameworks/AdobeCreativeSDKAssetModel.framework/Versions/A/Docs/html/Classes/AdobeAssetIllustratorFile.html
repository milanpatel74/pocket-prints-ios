Overview

AdobeAssetIllustratorFile represents an Adobe Illustrator Document (AI file). It provides methods for creating Illustrator files.

Parameters

name

The name of the file to create.

folder

The name of the AdobeAssetFolder containing the AI file to be created.

The AGC manifest used to create the AI file on Adobe Creative Cloud.

imageComponents

An array of AdobeAGCImageComponent objects.

overwrite

Whether to allow overwriting an existing destination file.

progressBlock

Optionally, track the upload progress.

successBlock

Optionally, get an updated reference to the created file when complete.

cancellationBlock

Optionally, be notified of a cancellation on upload.

errorBlock

Optionally, be notified of an error.

Parameters

name

The name of the file to create.

folder

The name of the AdobeAssetFolder containing the AI file to be created.

images

Array of AdobeAssetFile objects for the creation of the AdobeAssetIllustratorFile. These objects have to be SVG files without external references that may contain base64 encoded JPEG or PNG images. (Currently only the first array object is used).

overwrite

Whether to allow overwriting an existing destination file. Set to NO if the file doesn’t exist.

progressBlock

Optionally, track the upload progress.

successBlock

Optionally, get an updated reference to the created file when complete.

cancellationBlock

Optionally, be notified of a cancellation on upload.

errorBlock

Optionally, be notified of an error.
