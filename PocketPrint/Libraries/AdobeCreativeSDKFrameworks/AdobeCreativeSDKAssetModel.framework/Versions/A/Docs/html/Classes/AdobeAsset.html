Overview

AdobeAsset is the base class of an object that is on the Adobe Creative Cloud. It is extended by AdobeAssetFile, AdobeAssetFolder, AdobeAssetPSDFile, AdobeAssetSketchbook, AdobeAssetLineFile, AdobeAssetDrawFile, and AdobeAssetLibrary classes.
