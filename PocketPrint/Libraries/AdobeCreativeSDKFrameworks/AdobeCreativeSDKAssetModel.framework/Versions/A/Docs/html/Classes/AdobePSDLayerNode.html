Inherits from NSObject
Conforms to NSMutableCopying

Overview

AdobePSDLayerNode represents a layer node (either a leaf layer or a layer group within a PSD file. Notice that this is an immutable class. If you need to make changes to a layer node you will need to create a mutable copy of it and then call updateLayerNode: on its composite.

The following properties/methods have been deprecated:

  • bounds

The absolute index of the layer node within the structure of the PSD. The bottom-most layer (in z-order) has the lowest layerIndex – albeit not necessarily a specific value like 0 or 1. Is NSNotFound if the layer node is not part of a PSD composite branch.
