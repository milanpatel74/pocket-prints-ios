Adobe360ApplicationProperties Class Reference

Inherits from NSObject
Declared in Adobe360ApplicationProperties.h

Overview

A data model used to contain the necessary application properties needed for the 360 workflow transport layer.

These can be based on build time parameters you supply to your project or design items like accent color.
